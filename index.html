配置Tomcat

  2. 配置环境变量

  3. cmd—>service install Tomcat9

  4. 启动Tomcat，startup.bat

MarkDown语法

MarkDown学习

工具：Typora

教程：菜鸟教程

  1. 标题

    • 使用”#”标记标题, #的数量代表标记级数

      example：

      #+空格+“文字”

      我是一级标题

      #+空格+“文字”

      我是二级标题

  2. 格式

    • 字体

      • 斜体 （* 文字 *）
      • 粗体 （** 文字 **）

    • 分割线

      • 三个*

    • 删除线

      • 删了哦 （~ 文字 ~）

    • 下划线

      • 我有下划线 ( 文字 )

    • 脚注

      • 我有脚注 [^J]

      [^J]: 脚注的方式…你猜

  3. 列表

    • 无序
      • 标记：* 或 + 或 -
    • 有序
      • 数字+点+空格

  4. 区块

    • “>”

      我是引用区块

  5. 代码

    • 三个反引号（~键）

  6. 链接

    • 名称[ 链接名 ] ( 地址 )

      这是一个连接myblog

  7. 图片

    • ! [ 图片的替代文字 ] ( 图片地址 )

      ! 图标

    • 参考 how to add a pic?

  8. 表格

    • emmm不好展示，参考链接吧表格

  9. 高级技巧（暂时了解）

    • 支持HTML元素
    • 转义符 \
    • 公式

PS：typora一些的操作

  • Ctrl + [ 减少缩进
  • Ctrl + ] 增加缩进
